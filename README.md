# 삼성전자 OPI 주식보상 시뮬레이터

삼성전자 OPI(성과급) 중 주식보상 비율을 최적화하기 위한 웹 기반 시뮬레이터입니다.

## 주식보상 제도 요약

| 항목 | 내용 |
|------|------|
| 선택 범위 | 세전 OPI 지급액의 0~50% (10% 단위) |
| 추가 혜택 | 주식보상 금액의 **15% 추가 지급** |
| 보유 조건 | 1년 의무 보유 |
| 지급 방식 | 현금은 급여계좌, 주식은 증권계좌 |

## 실행 방법

### 단일 파일 버전 (권장)

```bash
# 서버 없이 바로 실행
open opi-simulation.html
```

`opi-simulation.html` 파일 하나로 모든 기능이 동작합니다. 더블클릭으로 바로 실행 가능.

### 멀티 파일 버전

```bash
# 방법 1: 직접 파일 열기
open index.html

# 방법 2: 로컬 서버 실행
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080 접속
```

## 입력 항목

| 입력 | 설명 | 예시 |
|------|------|------|
| 계약연봉 | 세전 연봉 전체 금액 | 150,000,000원 |
| 예상 OPI 비율 | 회사 성과에 따른 OPI 비율 (0~50%) | 25% |
| 기준주가 | OPI 지급 시점의 삼성전자 주가 | 55,000원 |
| 미래 예상 주가 | 1년 후 예상 주가 | 70,000원 |

## 주요 기능

- **비율별 비교**: 0%, 10%, 20%, 30%, 40%, 50% 각 비율의 상세 결과
- **주가 시나리오 분석**: 주가 변동(-30%~+30%)에 따른 최적 비율 변화
- **세금 영향 분석**: 현금 vs 주식 선택 시 세금 차이 (4대보험 + 소득세 누진세율)
- **손익분기점 분석**: 주식 선택이 유리해지는 최소 주가
- **다크모드**: 시스템 설정 연동
- **반응형 디자인**: 모바일/태블릿/데스크톱 지원

## 계산 로직

### OPI 주식보상 계산

```
OPI 지급액 = 계약연봉 × OPI비율
주식보상액 = OPI지급액 × 주식선택비율
추가혜택 = 주식보상액 × 15%
총 주식보상 = 주식보상액 + 추가혜택
지급주식수 = floor(총 주식보상 ÷ 기준주가)
차액 = 총 주식보상 - (지급주식수 × 기준주가)
```

### 계산 예시

```
입력: OPI 1,500만원, 주식보상 30%, 기준주가 11만원

주식보상액: 1,500만 × 30% = 450만원
추가혜택: 450만 × 15% = 67.5만원
총 주식보상: 450만 + 67.5만 = 517.5만원
지급주식수: floor(517.5만 ÷ 11만) = 47주
차액: 517.5만 - (47 × 11만) = 5,000원
```

| ~3억원 | 38% | 1,994만원 |
| ~5억원 | 40% | 2,594만원 |
| ~10억원 | 42% | 3,594만원 |

+ 지방소득세: 소득세의 10%

### 손익분기점

15% 추가혜택으로 인해 주가가 약 **13% 하락**해도 현금 수령과 동일한 가치를 유지합니다.

```
손익분기 주가 = 기준주가 × (1 - 0.15/1.15) ≈ 기준주가 × 87%
```
